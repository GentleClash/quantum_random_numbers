# quantum_random_numbers



### For Linux

```bash
git clone https://github.com/GentleClash/quantum_random_numbers.git
cd quantum_random_numbers
python -m venv qrng && source qrng/bin/activate
pip install -r requirements.txt
python main.py
```

### For Windows

```powershell
git clone https://github.com/GentleClash/quantum_random_numbers.git
cd quantum_random_numbers
python -m venv qrng
.\qrng\Scripts\activate
pip install -r requirements.txt
python main.py
```
